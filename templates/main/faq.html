Frequently Asked Questions

How do I start using Yuan Accounting?

To get started:

  1. Create an account or log in if you already have one
  2. Set up your accounts (e.g., cash, credit cards, bank accounts)
  3. Add your income and expense categories
  4. Start recording your transactions

How do I add a new account?

Go to the Accounts page and click the "Add Account" button. Fill in the account details including:

  • Account name
  • Initial balance
  • Currency
  • Account type (e.g., checking, savings, credit card)
How do I transfer money between accounts?

Use the Transfer function to move money between accounts:

  1. Click the "Transfer" button
  2. Select source and destination accounts
  3. Enter the transfer amount
  4. Add a description (optional)
  5. Click "Submit"

How do I organize my transactions?

Use categories to organize your transactions:

  • Create income categories for different revenue sources
  • Create expense categories for different types of spending
  • Assign icons to categories for easy recognition
  • Use the search and filter functions to find specific transactions
Can I edit or delete transactions?

Yes, you can edit or delete any transaction by:

  1. Finding the transaction in your records
  2. Clicking the edit (pencil) or delete (trash) icon
  3. Confirming your changes

What reports are available?

Yuan Accounting provides several reports:

  • Monthly income and expense summary
  • Category-wise spending analysis
  • Account balance trends
  • Cash flow statements
How can I export my data?

You can export your data in several formats:

  • CSV files for spreadsheet applications
  • PDF reports for printing or sharing
  • Detailed transaction histories

How can I get help?

There are several ways to get support:

  • Check this FAQ section for common questions
  • Contact our support team via email
  • Visit our documentation for detailed guides
  • Join our community forum for user discussions
Is my data secure?

Yes, we take security seriously:

  • All data is encrypted in transit and at rest
  • Regular security audits and updates
  • Secure authentication and authorization
  • Regular backups to prevent data loss
